# 更多更多语言 (More More Lang)

## 简介

厌倦了物品名称只显示您当前的游戏语言吗？ **More More Lang** 解决了这个问题，它能将您选择的任何语言的物品、方块和流体名称直接注入到它们的工具提示（tooltip）中。

对于那些经常与多语言社区交流、分享截图，或者只是想知道另一种语言（例如，即使您使用日文游玩，也想知道英文）中使用的“标准”名称的玩家来说，此 Mod 是必不可少的。

## 🚀 主要特性

*   **多语言显示：** 配置并同时显示物品在*任意数量*语言中的译名。
*   **自由选择：** 您不局限于只选择一种辅助语言；您可以选择您想看到的特定语言代码（例如，同时选择 `es_es` 和 `zh_cn`）。
*   **无缝集成：** 本地化的名称会整洁地显示在物品工具提示中的标准名称下方。
*   **兼容 Jade：** 与 [Jade](https://github.com/Snownee/Jade) Mod 完全且稳定地兼容，确保本地化名称能正确显示在高级工具提示界面中。
*   **轻量化：** 设计为高性能运行，对游戏加载时间的影响微乎其微。

## ⚙️ 配置方法

要显示的语言列表是通过 Mod 的配置文件设置的，该文件通常位于 `config/MoreMoreLang.toml`。

### 配置示例

您需要提供一个 ISO 语言代码列表（例如：`en_us`、`ja_jp`、`zh_cn`、`fr_fr`）。
